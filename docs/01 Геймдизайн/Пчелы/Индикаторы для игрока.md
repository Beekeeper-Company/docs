### Состояние семьи

| Параметр         | Что видит игрок (пример) | Визуализация                          | Комментарий                                       |
| ---------------- | ------------------------ | ------------------------------------- | ------------------------------------------------- |
| Сила семьи       | 4/5                      | Полоска с заполнением                 | суммарный показатель здоровья и продуктивности    |
| Популяция        | 12 340 пчел              | Число + иконка                        | Сколько всего пчел в улье                         |
| Рабочие пчелы    | 10 200                   | Число + иконка (иконка рабочей пчелы) | Основная рабочая сила. Следить за динамикой       |
| Трутни           | 340                      | Число + иконка (иконка трутня)        | Есть ли избыток. Осенью должны обнуляться         |
| Расплод          | 4 500                    | Число + иконка (иконка личинки)       | Будущее семьи. Если мало - проблемы с маткой      |
| Мед в улье       | 18,4 кг                  | Число + иконка (банка меда)           | Общий запас. Критичный показатель для зимы        |
| Перга            | 2,3 кг                   | Число + иконка (цветочная пыльца)     | Корм для расплода. Без нее личинки не будут расти |
| Свободные ячейки | 1200                     | Число + иконка  (пустые слоты)        | Место для расширения. Если 0 - нужны новые рамки  |
| Возраст матки    | 2 года                   | Число + иконка (корона)               | Пора ли менять                                    |

### Детализация популяции

| Параметр             | Что видит игрок                  | Визуализация     | Комментарий                                          |
| -------------------- | -------------------------------- | ---------------- | ---------------------------------------------------- |
| **Внутренние пчелы** | Общее количество, Число + иконка | Иконка улья      | Работают внутри улья. Основа семьи                   |
| - уборщицы           | Число + иконка                   | Веник            | Чистят ячейки. Если мало - грязь, болезни            |
| - кормилицы          | Число + иконка                   | соска            | Кормят расплод. Если мало - расплод голодает         |
| - строительницы      | Число + иконка                   | молоток          | Строят соты. если мало - нет новых ячеек             |
| - приемщицы          | Число + иконка                   | руки             | Принимают нектар. Если мало - затор у входа          |
| - вентиляторщицы     | Число + иконка                   | вентилятор       | Охлаждают/сушат. В жару критично                     |
| - охранницы          | Число + иконка                   | щит              | Защищают вход. Если мало - воровство                 |
| **Фуражиры**         | Общее количество, Число + иконка | Крыло            | Летают за ресурсами. Главные добытчики               |
| - нектарные          | Число + иконка                   | капля меда       | Собирают нектар. Основа медосбора                    |
| - пыльцевые          | Число + иконка                   | цветок с пыльцой | Собирают пыльцу. Нужны для создания перги            |
| - водоносы           | Число + иконка                   | капля воды       | Носят воду. В жару спасают улей                      |
| - прополисные        | Число + иконка                   | смола            | Собирают прополис. Заклеивают щели                   |
| - разведчицы         | Число + иконка                   | глаз             | Ищут новые места. Опрелеояют эффективность медосбора |

Концепция - интерфейс как часть мира. Вся информация, которую игрок получает о мире, должна быть его частью. Нет HUD-а, минимальный интерфейс появляется только при взаимодействии с объектом. Все меню существуют физически в мире игры.

### HUD - только при взаимодействии
   - кнопки-подсказки при взаимодействии с объектом
   - названия объектов, с которыми можно взаимодействовать
   - иконка действия рядом с названием
   - возможность открыть справку с разбором механик при взаимодействии с объектом

##### Улей
Минимальные индикаторы, когда открыт улей: количество пчел, наполнение рамок медом, векторные иконки в одном стиле, что-то вроде:
─────────────────────────────────────
[ИКОНКА ПЧЕЛЫ]  24 / 30  |  [🍯]  8/10
─────────────────────────────────────
##### Уведомления
В правом верхнем углу появляются короткие сообщения типа "Мед собран +3", "Новая цель: проверить медоносы", "Пчелы забеспокоились!". Исчезают через 2 секунды



### Книга знаний (кодекс), журнал с квестами, индикаторы состояния пчел
В мастерской на столе лежит старая потрепаная книга. Игрок подходит, нажимает Е - открывается интерфейс книги. Нужна 3д модель.
В книге переворачиваются страницы, на страницах различная информация типа:

┌──────────────┐ 
│                                    |   
│  ГЛАВА 1: ПЧЕЛЫ     |   
│  • Типы пчел             │ 
│  • Здоровье              │
│  • Размножение       |
│                                    │
│  ГЛАВА 2: ЦВЕТЫ     |
│  • Луговые                │  
│  • Лесные                  │
│  • Редкие                   │ 
│                                     │
└──────────────┘  
Шрифт рукописный. страницы стилизованы под бумагу, рисунки/наброски пчел и цветов как в старых энциклопедиях, анимация переворотов страниц, уголки страниц загнуты/потерты, торчит закладка (быстрая навигация).

Функциональные разделы книги:
1. Пчелы: энциклопедия типов пчел, их параметры, болезни
2. Цветы: какие цветы где растут, какую пыльцу дают
3. Оборудование: описание ульев, инструментов, улучшений
4. Заметки деда: история, лор, советы (нарратив)
5. Квесты/цели: заметки персонажа (персонаж планирует себе цели, отмечает прогресс)
6. Статистика: сколько меда собрано, дней прошло и так далее

Необходимо сделать:
- шаблон страницы (фоновый слой с текстурой)
- набор иконок для всех типов пчел, цветов, ресурсов
- шрифты: основной рукописный, дополнительный печатный
- элементы: заголовки, буквицы, разделители, рамки для иллюстраций
- анимация перелистывания
### Инвентарь - зона хранения в мастерской
У игрока нет портативного инвентаря, он может носить с собой ограниченное количество инструментов (на поясе, в руках).
Основной инвентарь:
- ящик с инструментами в мастерской: для мелких предметов (инструменты, расходники)
- стеллаж с банками: для хранения меда и продуктов
- верстак: для крафта и улучшений

Ящик.
Физический объект в мастерской. При взаимодействии открывается UI, стилизованый под открытый ящик. Отделения ящика визуально разделены деревянными перегородками. Предметы лежат как в настоящем ящике (немного под углом, в беспорядке). При наведении показывают название.

Стеллаж.
Игрок подходит к стеллажу, видит полки с банками меда. При взаимодействии камера приближается, и игрок может:
- посмотреть сколько меда каждого сорта (подписаны банки, интерфейс показывает количество)
- взять банку (для действий вроде продажи и перемещения)
- поставить банку (сохранить)
Визуально это реальные 3Д-банки на полках, а не иконки. Количество меда можно определить количеством банок и уровнем наполнения. Также количество меда подсчитывается в журнале.

Верстак для крафта.
Стол с инструментами и чертежами. UI открывается как разложенный чертеж. Чертеж выглядит как старая схема с рукописными пометками. 

### Иконки
Векторные иконки с тонкими линиями (1.5-2 пкс), скругленные углы, в едином стиле:
- пчела (общая)
- рабочая пчела/трутень/королева
- специализации рабочих пчел (есть в списке)
- воск/пыльца/прополис
- цветы (разные виды)
- инструменты: дымарь, стамеска, щетка
- монета/продажа
- часы/время
- погода (солнце/дождь/ветер)
- здоровье/болезнь
- квест
- книга/кодекс

### Критерии готовности вертикального среза
- [ ] Все Must Have задачи из эпиков 1-5 выполнены
- [ ] Игра запускается без ошибок
- [ ] Можно пройти полный цикл Core Loop
- [ ] Есть понятная цель
- [ ] Мир выглядит целостно
- [ ] Звук есть и не раздражает
- [ ] Протестировано на 2-3 людях не из команды
- [ ] Собран фидбек и запланированы правки

Демо-версия:
- [ ] собран билд игры
- [ ] файл запускается
- [ ] написана инструкция к игре
- [ ] презентация о процессе разработки
- [ ] геймплейное видео для оформления демки

Ретроспектива вертикального среза
- [ ] что получилось хорошо
- [ ] что получилось хуево
- [ ] что будем делать иначе в следующий раз
- [ ] что хотим добавить в полную версию
